A WordPress plugin to analyze mood of post content for positive, negative, or neutral.

== Description ==

Content Mood Analyzer is a powerful WordPress plugin that analyzes the mood of your post content. It determines whether the content is positive, negative, or neutral based on keyword analysis.

Features:
- Analyze individual posts
- Bulk analysis of multiple posts
- Display mood badges on posts
- Customizable keywords for each mood type
- REST API endpoints for integration
- Admin dashboard for management

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/content-mood-analyzer` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the Settings->Content Mood Analyzer screen to configure the plugin.

== Frequently Asked Questions ==

= How does the analysis work? =

The plugin uses keyword matching to determine the mood. It counts occurrences of positive, negative, and neutral keywords in the post content.

= Can I customize the keywords? =

Yes, you can add or modify the keywords for each mood type in the plugin settings.
